## What is it?

SolanaMemeDeployer lets you create Solana SPL tokens without writing code or using the command line. Connect your wallet, fill in your token details, upload a logo, and deploy. Your token is live on chain in about 30 seconds.

## Features

- **Full token creation** in a single transaction
- **Metaplex metadata** (name, symbol, logo visible in wallets and explorers)
- **Automatic mint authority revocation** (locked supply, fully verifiable)
- **Client side signing** (your keys never leave your wallet)
- **0.4 SOL flat fee** with no hidden upsells

## How it works

1. Connect your Phantom or Solflare wallet
2. Enter token name, symbol, decimals, and supply
3. Upload your logo
4. Click deploy
5. Done. Token is live.

- React + Vite
- Solana Wallet Adapter
